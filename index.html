How to get your website onto the Internet using Github

First, save your html file and any additional files.

Second, right click in the folder you have your files saved into and select Git Bash.

Third, type the command $ git init

Fourth, type the command $ git checkout -b gh-pages. This changes the branch of your files

Fifth, type the command $ git status to see where you are in the process of getting your website to Github.

Sixth, type the command $ git add filename or $ git add . if you want to add everything in the folder.

Seventh, check to see if you added the files with $ git status

Eighth, type the command $ git commit -m "message". Make sure to write a good description of the commit you are about to make.

Nineth, check to make sure the files are commited with $ git status. It should say that it is working the directory clean.

Tenth, Sign into Github on an Internet browser and add a repository. It should then give you two lines of code in the second section. Copy only the first line and insert it into the command screen

Eleventh, type the command $ git push -u origin gh-pages. It then shuld ask you your Github login information. The password will not be shown. Once logged in, return back to Github and refresh the page. Check to see if all of your files made it to here.

To see if your website made it to github type the URL http://username.github.io/websitename
